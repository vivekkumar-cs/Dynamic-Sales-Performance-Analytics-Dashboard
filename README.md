# Dynamic Sales Performance Analytics Dashboard

## 📊 Project Overview
This project is a comprehensive Excel-based data analytics solution designed to transform raw sales records into actionable business insights. Using a macro-enabled workbook (`.xlsm`), it provides a high-level view of sales operations and key performance indicators.

## 🚀 Key Features
* [cite_start]**Automated Data Processing:** Utilizes VBA macros (stored in `vbaProject.bin`) for workflow automation[cite: 13].
* [cite_start]**Interactive Data Exploration:** Integrated **Slicers** allow for real-time filtering of sales metrics[cite: 11].
* [cite_start]**Multi-Pivot Analysis:** Powered by **four distinct Pivot Tables** to aggregate complex datasets into digestible summaries.
* [cite_start]**Visual Insights:** Includes multiple customized **Charts** (Trend, Category, and Style-specific) to identify patterns at a glance[cite: 11, 12, 13].

## 🛠️ Technical Stack
* **Microsoft Excel:** Core engine for data processing and visualization.
* [cite_start]**VBA (Visual Basic for Applications):** For backend automation and custom logic[cite: 13].
* [cite_start]**Pivot Modeling:** Advanced data grouping and calculation[cite: 14].
* [cite_start]**XML-based Spreadsheet Structure:** Optimized for professional data portability[cite: 1, 15].

## 📂 File Structure
* `DASHBOARD sales data.xlsm`: The primary macro-enabled workbook containing the dashboard, raw data, and pivot calculations.

## 📝 How to Use
1. **Clone the Repo:** `git clone https://github.com/vivekkumar-cs/Dynamic-Sales-Performance-Analytics-Dashboard.git`
2. **Open the File:** Use Microsoft Excel to open `DASHBOARD sales data.xlsm`.
3. **Enable Macros:** Click "Enable Content" when prompted to ensure all automated features function correctly.
4. **Interact:** Use the on-screen slicers to filter data by various dimensions.
